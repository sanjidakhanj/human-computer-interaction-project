# Human–Computer Interaction (HCI) Project

## Description
This is an academic Human–Computer Interaction coursework project
focused on user-centered design and usability principles.

## Key Topics
- User research and personas
- Task analysis and scenarios
- Interface design and wireframes
- Usability evaluation
- HCI principles and heuristics

## Tools / Methods
- User-centered design
- Usability testing
- Wireframing and prototyping


## Notes
This project was completed as part of a university HCI module.
